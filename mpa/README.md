# MPA for Linux

Linuxでマイナポータルにログインするためのブラウザ拡張およびNativeMessagingホストアプリケーションです。

下記のサイト&ログイン方式を確認しています。
- マイナポータルでICカードリーダライタを使ってログイン
- e-taxでマイナンバーカードをICカードリーダーで読み取りでログイン

## 概要

マイナポータルにカードリーダー方式でログインするにはWindows/macOSであれば
- マイナポータルブラウザ拡張
- マイナポータルAP(ホストアプリケーション)
の2つをインストールします。

であればマイナポータルAP(MPA.exe)のLinux版のみ作ればよさそうですが、
MPA.exe互換のホストアプリケーションを作ったとて、
navigator.userAgentなどを見て弾かれてしまいます。
したがってブラウザ拡張とホストアプリケーションの両方が必要となります。

## ホストアプリケーションのインストール

このディレクトリで
```
cargo install --path .
```

もしくはプロジェクトrootで

```
cargo install --path mpa
```

`~/.cargo/bin/mpa`にインストールされます。

## ホストアプリケーションの登録

```
./install.sh
```

ChromeのProfileを指定する場合

```
./install.sh --user-data-dir /path/to/datadir
```

## ブラウザ拡張のインストール

- `chrome://extensions/`を開く
- 右上のディベロッパーモードをON
-「パッケージ化されていない拡張機能を読み込む」で`./mpa/extension`を読み込む
- 拡張機能のメニューから「MPA for Linux」を開く
- 動作確認ボタンを押してエラーが出なければOK

## ログイン

ログイン画面で暗証番号入力ウィンドウが表示されます。
カードを設置した上でJPKI認証用の暗証番号(4桁)を入力
